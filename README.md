# Projet de tableau de bord – Mode de vie et cancers

## Description du projet
Ce projet a été réalisé dans le cadre d’un **projet universitaire**.  
Il vise à analyser le **lien entre certains facteurs liés au mode de vie et le développement de différents types de cancers**, à partir d’articles scientifiques issus de **PubMed**.

Les types de cancers étudiés incluent :
- Cancer du poumon
- Cancer du sein
- Cancer du côlon / colorectal
- Cancer de la prostate

Les facteurs analysés concernent des dimensions **socioéconomiques, comportementales et environnementales**, telles que le revenu, le stress, le tabagisme, l’alimentation ou le niveau d’éducation.

---

## Méthodologie
1. **Recherche bibliographique** sur PubMed à l’aide d’une équation de recherche ciblée (2000–2025).
2. **Collecte des données** via des scripts **Python** (web scraping et extraction des métadonnées des articles).
3. **Création et alimentation d’une base de données Oracle** :
   - Conception du modèle relationnel
   - Création des tables (articles, cancers, facteurs, années, pays, etc.)
   - Nettoyage et structuration des données
   - Export et stockage des données intermédiaires au format CSV
4. **Analyse et visualisation** des données avec **Power BI** à travers un tableau de bord interactif.

---

## Objectifs du tableau de bord
- Étudier la distribution des articles par type de cancer
- Identifier les facteurs de mode de vie les plus fréquemment associés
- Analyser l’évolution temporelle des publications
- Comparer les facteurs selon les types de cancers
